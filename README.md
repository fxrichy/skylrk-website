# SKYLRK - Premium Website Clone

A modern, dark-themed website built with React and Tailwind CSS, inspired by the skylrk.com design aesthetic.

## Features

- **Dark Theme**: Sleek black background with subtle gradients
- **Modern Typography**: Inter font family with excellent hierarchy
- **Responsive Design**: Mobile-first approach, works on all devices
- **Smooth Animations**: Framer Motion ready (integrated)
- **Component Architecture**: Modular React components

## Sections

1. **Navigation** - Fixed header with blur effect on scroll
2. **Hero** - Large typography with CTA buttons
3. **Features** - 4-column grid with icons
4. **Gallery** - Project showcase grid
5. **Contact** - Contact form with info
6. **Footer** - Multi-column links with social icons

## Installation

```bash
# Install dependencies
npm install

# Add required packages
npm install framer-motion lucide-react

# Start development server
npm start
```

## Build for Production

```bash
npm run build
```

## Tech Stack

- React 18
- Tailwind CSS
- Framer Motion
- Lucide React Icons
- Inter Font (Google Fonts)

## Design Tokens

- **Background**: #000000 (black)
- **Text**: #ffffff (white)
- **Accent**: #8BC4E4 (light blue)
- **Borders**: rgba(255,255,255,0.05-0.2)
